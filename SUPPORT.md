# Support

This release is an experimental alpha with community support only.

Before requesting help:

1. Read `README.md`, `docs/USER_GUIDE.md`, and `docs/TROUBLESHOOTING.md`.
2. Run the installer in verification mode.
3. Record DrumSLICE ID, Live, Max, Python, OS, and architecture versions.
4. Reproduce the problem with the smallest rack structure possible.
5. Review diagnostics for private source paths before sharing them.

Use GitHub Discussions for setup and workflow questions and the bug template for reproducible defects. Do not upload copyrighted sample packs or Live Sets you cannot redistribute. Use the private process in `SECURITY.md` for vulnerabilities.
