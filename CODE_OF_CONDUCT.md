# Code of conduct

Be respectful, specific, and constructive. Harassment, discrimination, threats, sexualized attention, deliberate disruption, and publication of another person's private information are not acceptable in project spaces.

Maintainers may edit or remove contributions that violate these expectations and may temporarily or permanently restrict participation. Report conduct concerns privately through the repository maintainer's GitHub profile; do not use a public issue to disclose sensitive personal information.

This policy applies to repository issues, pull requests, discussions, and other spaces where someone represents the project.
